# Alunos POO
Um "projeto" para por em prática os aprendizados sobre POO no C++

## Como rodar
No cmd (com o compilador instalado): <br />
**Linux:** <br />
`code
g++ *.cpp -o main.out
`
<br />
`code
./main.out
`
<br />
<br />
**Windows:** <br />
`code
g++ *.cpp -o main.exe
`
<br />
`code
./main.exe
`
<br />
<br />
Caso o código `g++ *.cpp -o main.exe` não funcione, tente usar outro prompt do Windows.
